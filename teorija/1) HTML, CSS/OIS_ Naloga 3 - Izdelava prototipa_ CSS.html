CSS

Na tretjih vajah se bomo seznanili z jezikom za oblikovanje spletnih strani - CSS (Cascading Style Sheets).

Naloga:

Obstoječe spletne strani opremite s stilnimi predlogami ter jih oblikujte. Poskrbite za stilno preobrazbo strani, tako da spremenite barve ozadja, pisavo, lepše oblikujete slike, besedila, povezave, nastavite razmake med elementi ipd. Besedilo, ki ste ga do zdaj oblikovali s pomočjo html značk (b, i, h1, ...) sedaj oblikujte s pomočjo CSS. Pri tem uporabite različne načine vključitve CSS v HTML, več vrst selektorjev in čim več različnih CSS lastnosti. Pri iskanju relevantnih lastnosti si pomagajte s stranjo W3 Schools. Stran vsebuje tudi tutoriale za vse ostale dele CSS in jo toplo priporočam kot pomoč pri reševanju naloge.

Zahteve:

  • Obvezno preizkusite 3 različne načine vključevanja CSS v html dokumente:
    • Sprotni zapis CSS (Inline CSS) - minimalno pri 1 elementu
    • Vgrajen zapis CSS (Internal CSS) - pri 3 ali več elementih (z enim zapisom CSS vplivajte na slog 3 ali več HTML elementov)
    • Zunanji zapis CSS (External CSS) - uporabite za vse ostale stilske spremembe.
      • Uporabite eno CSS datoko za vse strani!
  • Pri nalogi obvezno uporabite naslednje selektorje:
    • element selector (npr. p, h1, h2, table, body itd.) s katerim označite skupino elementov HTML, ki pripadajo isti znački in jih ustrezno stilizirate,
      • Pri tem dodatno demonstrirajte uporabo več »element selector«-jev hkrati (npr. določite stile, ki veljajo za h1 in h2).
    • class selector s katerim označite skupino elementov HTML z ustreznim atributom class in jih ustrezno stilizirate,
      • Pri tem dodatno demonstrirajte kombiniranje stilov ter uporabo class selektorja pri elementih, ki imajo več kot en stil (npr. pripadajo dvema razredoma »center large« in še kakšnemu id-ju).
    • id selektor s katerim označite element HTML z ustreznim atributom id in ga ustrezno stilizirate,
      • Ne pozabite, da morajo biti id selektorji unikatni.
    • Kombinirani in gnezedeni selektorji (npr. div > p; div p, tr:nth-child(), ...).
  • V sklopu stilne preorazbe spremenite vsaj:
    • Značke b, i ,h1, h2, ... zamenjajte s css in ustreznimi selektorji
    • barvo, velikost besedil ter nastavite ustrezen font,
      • Definicija barv, pisav ipd. je poljubna. Pomagate si lahko s spletno storitvijo https://coolors.co/.
      • Fonte, ki so že vključeni v HTML standard (jih ni potrebno uvoziti) lahko najdete na strani W3 schools - CSS Fonts
      • Lahko uporabite tudi zunanje fonte, ki jih najdetena na straneh kot je Google Fonts, vendar jih morate potem ustrezno uvoziti.
    • barvo ozadja,
    • obrobe tabel,
    • barvo gumov,
    • lastnosti margin in padding,
    • povezave naj ob prekritju z miško spremenijo barvo,
    • slike naj ob prekritju z miško postanejo delno prosojne.
  • Poskrbite za enoten izgled vseh strani, še posebej noge (header) ter glave (footer).
  • Poskrbite za lepšo postavitev strani z uporabo div ali html5 elementov.

    • 3 točke
      • Študent je pozabil (ali napačno uporabil) spremeniti največ 3 nujne elemente za preobrazbo ter največ 1 skupino selektorjev.
    • 4 točke

    Kaj je CSS?

    Stilne predloge (CSS) na spletni strani

    Stilne predloge ali CSS (ang. Cascading Style Sheets) uporabimo, ko želimo na spletni strani oz. spletišču (več povezanih spletnih strani) enotno oblikovati izgled strani (naslove, menije, povezave, besedilo v tabelah, ...).

    Veliko stvari, ki jih dosežemo z uporabo CSS, bi sicer lahko dosegli tudi z oblikovanjem znotraj samega dokumenta HTML (npr. oblikovanje pisave za naslove), vendar je uporaba CSS bistveno bolj učinkovita. Če bi na spletnih straneh imeli npr. 20 naslovov, bi morali znotraj HTML 20-krat nastaviti enake lastnosti; zelo verjetno je, da bi pri tem pozabili na kakšno podrobnost. Še na večjo težavo bi naleteli, če bi kasneje želeli spremeniti katero izmed teh lastnosti; recimo, da bi želeli zamenjati le barvni odtenek - povsod, kjer se pojavljajo naslovi, bi morali ročno spreminjati to lastnost. Z uporabo CSS dosežemo poenoten način oblikovanja - vse lastnosti nastavimo le na enem samem mestu; npr. ob želeni spremembi naslova spremenimo le CSS in vse spletne strani se bodo avtomatsko prilagodile tej spremembi brez spreminjanja dokumentov HTML. Oblike nekaterih elementov (npr. oblike povezav) pa brez uporabe CSS sploh ni mogoče spremeniti.

    Na spodnji sliki vidimo isto spletno stran, pri kateri smo le zamenjali pripadajoč CSS - razlika v obliki je več kot očitna.


    Prednosti uporabe CSS

    Z uporabo CSS imamo kar nekaj bistvenih prednosti:

    • izgled ene ali več spletnih strani lahko spreminjamo z urejanjem enega samega dokumenta (dokumenta CSS)
    • koda HTML je bolj čista in pregledna ter lažje razumljiva, ker ne vsebuje več oznak za oblikovanje prikaza, ampak se osredotoči na samo vsebino
    • spletne strani je lažje vzdrževati
    • uporaba CSS omogoča boljši nadzor nad prikazovanjem spletnih strani kot sam HTML

    Pisanje CSS

    Osnovna elementa CSS sta lastnost in vrednost. Ločuje ju dvopičje, vrstico pa zaključimo s podpičjem. Imena lastnosti ne morejo vsebovati presledkom, zato po navadi vsebujejo pomišljaje. Za razliko od atributov pri HTML elementih, vrednosti CSS lastnosti niso v narekovajih.
    lastnost-elementa: vrednost;

    Za določanje pri katerih HTML elementih naj vrednost CSS lastnosti velja, uporabljamo selektorje, ki se pišejo tako:
    selektor {
        lastnost-elementa: vrednost;
    }

    Vključevanje CSS v spletno stran

    Sprotni zapis CSS (Inline CSS)

    CSS lahko dodamo na spletno stran na tri načine; prvi način je sprotni zapis (inline CSS), kjer zapišemo CSS v vrstici elementa HTML (kot atribut označbe HTML):

    <h1 style="color: #FF0000;">Naslov v rdeči barvi</h1>

    Tak zapis je seveda enakovreden naslednji kombinaciji označb HTML:

    <h1><font color="#FF0000">Naslov v rdeči barvi</font></h1>

    Vgrajen zapis CSS (Internal CSS)

    Drugi način dodajanja CSS na spletno stran je zapis v glavi dokumenta HTML z uporabo značke <style> - stil definiramo v glavi dokumenta HTML in ga kasneje v telesu uporabimo:

    <head>
  <style>
    h1 {color: red; }
    h2 {color: blue; }
  </style>
</head>

<body>
  <h1>Naslov v rdeči barvi...</h1>
  ...
</body>

    Vsak element (v zgornjem primeru naslova <h1> oz. <h2>), ki ga bomo uporabili kjerkoli v dokumentu HTML, bo avtomatsko prikazan na način, kot smo zapisali v glavi, ko smo stil definirali. Na ta način lahko definiramo poljubno število stilov hkrati.

    Zunanji zapis CSS (External CSS)

    Tretji način zapisa CSS je zunaj dokumenta HTML kot samostojna datoteka CSS; v datoteko CSS zapišemo stile na enak način, kot pri vgrajenem zapisu CSS v glavo dokumenta HTML. V glavi dokumenta HTML nato zapišemo le povezavo na stilno predlogo:

     h1 {color: red; }
 h2 {color: blue; }
 .pomembno { font-family: Arial, Helvetica ;
             font-size: 12pt ;
             color: #003366;
               }

    stilna_predloga.css

    <head>
  <link rel="stylesheet" href="stilna_predloga.css" />
</head>

    spletna_stran.html

    Prednost tovrstnega zapisa je, da lahko povezavo na zunanjo datoteko CSS naredimo v poljubnem dokumentu HTML - na ta način lahko uporabimo enake stile na poljubnem številu različnih spletnih strani.

    Razredi (Classes)

    Poleg tega, da lahko spreminjamo obstoječe označbe HTML (v zgornjem primeru <h1> oz. <h2>), lahko ustvarimo tudi posebne selektorje, imenovane razredi (Class). Le-te nato uporabimo v tistih primerih, ko to izrecno želimo:

    <head>
  <style>
     .pomembno { font-family: Arial, Helvetica;
       font-size: 22pt;
       color: red; }
  </style>
</head>

<body>
  <p>Običajno oblikovan odstavek...</p>
 
  <p class="pomembno">
    Pomemben odstavek...
  </p>

      Še en...
  <span class="pomembno"> ... pomemben ... </span>
  del
    
</body>

    Ime razreda pri definiciji stila zmeraj začnemo s piko, ki pa je pri uporabi razreda ne pišemo! Razred z nekim stilom lahko v dokumentu HTML uporabimo pri različnih elementih strani (za odstavek, celico tabele, le nekaj izbranih besed, ...)

    En element lahko pripada več razredom. Primer je prikazan spodaj.

    <head>
  <style>
     .pomembno { font-size : 22pt; }
         .rdece { color: red; }
         .modro { color: blue; }
         .podcrtano { text-decoration: underline; }
  </style>
</head>

<body>
  <p>Običajno oblikovan odstavek...</p>
 
  <p class="pomembno rdece">
    Pomemben odstavek...
  </p>

      Še en...
  <span class="pomembno modro podcrtano"> ... pomemben ... </span>
  del
    
</body>

    Identifikatorji (ID-ji)

    Podobno kot razrede, lahko za definicijo različnih stilov nad enakim HTML elementom uporabimo identifikatorje. Namen identifikatorjev je enak kot namen razredov, le da se označujejo z znakom '#', HTML elementu pa je potrebno dodati atribut 'id'. Vsak element lahko pripada samo enemu id in vsak id ima lahko samo en element (v eni html datoteki)!

    <head>
  <style>
     #pomembno { font-family : Arial, Helvetica ;
       font-size : 22pt ;
       color : #003366 ; }
  </style>
</head>

<body>
  <p>Običajno oblikovan odstavek.</p>

  <p id="pomembno">Pomemben odstavek.</p>

  <span id="pomembno">Pomemben del.</span> <!-- NAPAKA -->
</body>

    Kdaj uporabiti razred in kdaj identifikator?
    V HTML dokumentu lahko isti razred uporablja več različnih elementov, s čimer elemente razvrstimo v nekakšne skupine, medtem ko z identifikatorji elementom določamo enoličnost. Torej, kadar želimo posameznemu elementu določiti enolično obliko, elementu dodamo atribut 'id', kadar pa bi radi poenotili obliko nad več elementi hkrati, izberemo atribut 'class'.

    Selektor elementov (ang. Element selector)

    Izbere HTML elemente na podlagi njihovega imena (značke).

    V tem primeru smo spremenili barvo in poravnavo vseh elementov z značko h2:

     h2 {
        color: red;
        text-align: center

Le-te nato uporabimo v tistih primerih, ko to izrecno želimo. Selektor razreda zbere HTML elemente na podlagi razreda (ali več razredov), ki smo ga (jih) jim dodelili.     Id selektor (ang. Id selector)

    Podobno kot razrede, lahko za definicijo različnih stilov nad enakim HTML elementom uporabimo identifikatorje. Namen identifikatorjev je enak kot namen razredov, le da se označujejo z znakom '#', HTML elementu pa je potrebno dodati atribut 'id'. Vsak element lahko pripada samo enemu id in vsak id ima lahko samo en element (v eni html datoteki)!
    <h2 id="id-elementa">Naslov strani</h2>
    <p id="id-elementa">Odstavek</p> <!-- NAPAKA -->
    spletna_stran.html
    V CSS datoteki, se z uporabo razreda na HTML element nanašamo enako kot pri selektorju razredov, le da tu namesto "." dodamo znak "#", saj s tem povemo, da je selektor id in ne razred. Enako kot razred, id ne sme vsebovati presledkov, zato si pomagamo s pomišljaji "-" ali pa podčrtaji "_". Id elementa je lahko poljuben najpogosteje pa je nek semantično smiselek niz (npr.: vnosni-obrazec-nakup, glavni-naslov, tabela-študentov, ...)
    #id_elementa{
        color: red;
    }
    stilna_predloga.css

    Kdaj uporabiti razred in kdaj identifikator?
    V HTML dokumentu lahko isti razred uporablja več različnih elementov, s čimer elemente razvrstimo v nekakšne skupine, medtem ko z identifikatorji elementom določamo enoličnost. Torej, kadar želimo posameznemu elementu določiti enolično obliko, elementu dodamo atribut 'id', kadar pa bi radi poenotili obliko nad več elementi hkrati, izberemo atribut 'class'.

    Univerzalni selektor (ang. Universal selector)

    Univerzalni selektor "*" izbere vse elemente v HTML dokumentu. V spodnjem primeru smo na celi spletni strani spremenili barvo ozadja ter sredinsko poravnali besedilo.
    * {
    text-align: center;
    background-color: blue;
}
    stilna_predloga.css


    Kombiniranje oz. grupiranje selektorjev

    S pomočjo grupiranih selektorjev lahko nastavljamo slog več različnih elementom hkrati.
    Primer:
    h1 {
        text-align: center;
        color: red;
    }

    h2 {
    text-align: center;
        color: red;
    }

    p {
    text-align: center;
        color: red;
}

    V zgornjem odseku CSS nastavljamo vsem h1, h2 in p elementom iste lastnosti (sredinsko poravnavo in rdečo barvo besedila). Namesto, da vse to pišemo 3x, lahko uporabimo kobinirni selektor in CSS lastnosti napišemo le 1x. To storimo tako, da napišemo več selektorjev in jih ločimo z vejico ",". Napisan CSS bo tako veljal za vse elemente, ki ustrezajo enemu izmed selektorjev.
    Primer:
    h1, h2, p {
        text-align: center;
        color: red;
    }

    Pri kombiniranjem selektorjev pa lahko uporabimo tudi selektorje različnih vrst.
    Na primer:
    h1, .naslov, #id-elementa {
        text-align: center;
        color: red;
    }


    Te tri selektorje lahko med sabo tudi kombiniramo, da dobimo ven natančnejši selektor. Kombiniramo lahko ime elementa z razredom ali ime elementa z identifikatorjem. Poglejmo si dva primera.

    p.razred1 {
  ...
}

    V tem primeru bomo spreminjali izgled vseh elementov p (paragraf), ki so razreda razred1: <p class="razred1">.

    a#id1 {
  ...
}

    V tem primeru pa bomo spreminjali izgled elementa a (povezav), ki ima identifikator id1: <a id="id1">


    Gnezdenje selektorjev

    Če naštevamo elemente brez vejice, ima selektor drugačen pomen. V spodnjem primeru določamo stil vsem elementom a, ki so znotraj elementa p na katerem koli nivoju.

    
                        
                            
    
                                
    p a {
        ...
    }

    Lahko smo še bolj natančni. Z znakom > povemo, da želimo označiti samo tiste elemente, ki so neposredno v določenem elementu. V spodnjem primeru izberemo vse elemente a katerih neposredni nad element je p.

    
                        
                            
    
                                
    p > a {
        ...
    }

    Pri oblikovanju lahko upoštevamo tudi atribute html elementov. Dva primera sta prikazana spodaj. V prvem primeru oblikujemo vse elemente a, ki vsebujejo atribut href (neglede na vrednost tega atributa). V drugem primeru, pa ne samo, da element more vsebovati atribut href, temveč tudi njegova vrednost mora biti "http://www.wikipedia.org".

    a[href] {
        ...
    }
    a[href="http://www.wikipedia.org"] {
       ...
    }


    Prioritete CSS predlog

    Ali ste se kdaj vprašali, katere lastnosti bodo v brskalniku prikazane, kadar v spletno stran vključite več CSS predlog, ki definirajo enake lastnosti? Kaj pa če lastnost definirate tako v predlogi, ki je vključena v HTML element, in hkrati v zunanji stilni predlogi? Brskalniki pri tem upoštevajo določene prioritete, kjer imajo najvišjo prioriteto predloge, ki so vključene v HTML elemente, najnižjo prioriteto pa predloge, ki so v brskalniku privzete:

    1. Stilne predloge, vključene v HTML elemente 
    2. Interne stilne predloge ali zunanje stilne predloge - odvisno od zaporedja vključitve
    3. Privzete predloge brskalnika

    To pomeni, da se vedno najprej upoštevajo lastnosti v vključeni CSS predlogi ne glede na lastnosti v ostalih predlogah. Kako to zgleda v praksi, prikazuje naslednji primer. Besedilo je rdeče barve, ker ima vključena CSS predloga prednost pred interno predlogo.

    <html>
  <head>
    <style>
        .moj_razred{ color:blue; }
    </style>
  </head>
  <body>
    <span style="color:red;" class="moj_razred">
          Rdeča kapica
        </span>
  </body>
</html>
    		  ->
    		 Rdeča kapica


    Oblikovanje tabel ali seznamov

    Tabelo ali seznam lahko oblikujemo tako, da je vsaka vrstica ali vsak element seznama drugačnega stila. Spodaj je primer kjer lihe in sode vrstice tabele obarvamo drugačne barve.

    tr:nth-child(odd) { /* lihe vrstice */
  background-color: grey;
}

    tr:nth-child(even) { /* sode vrstice */
      background-color: blue;
    }

    Če želimo izbrati točno določeni element pa preprosto vnesemo zaporedno mesto tega elementa.

    li:nth-child(3) { /* tretji element */
  color: red;
}
    Pomembnejši CSS elementi

    Včasih smo za oblikovanje spletnih strani v celoti uporabljali tabele in okvirje (frame). Danes se ta način več ne uporablja, vse se oblikuje s CSS-om. Oblikovanje strani s CSS-om prinaša dodatne funkcionalnosti, lepšo preglednost in možnost enostavnega spreminjanja celotne spletne podobe iz ene datoteke. Sledijo najpomembnejši elementi za oblikovanje s CSS-om.

    Margin in Padding

    Margin predstavlja prazen prostor, ki ločuje rob elementa od drugih elementov. Padding predstavlja prazen prostor med robom elementa in vsebino elementa. Podobno kot pri robovih lahko tudi margin in padding definiramo le na posameznih straneh elementa: levo, desno, zgoraj in spodaj. Pri posamezni lastnosti pa lahko znotraj ene deklaracije ločeno nastavljamo vrednosti za vsako stran posebej, pri čemer je vrstni red sledeč: top, right, bottom, left. Podobno lahko storimo tudi, ko skupaj zapišemo lastnost za zgornjo in spodnjo oz. levo in desno stran elementa.

    margin-left: 20px;
    margin-bottom: 30px;
    padding-right: 5px;
    padding-top: 150px;
    margin: 20px 10px 5px 15px;  /* top right bottom left */
    padding: 20px 10px;  /* top bottom, right left */

    Kaj je razporeditev?

    Preden pričnemo z izdelavo spletne strani, je smiselno pripraviti načrt postavitve elementov, kot so glava in logotip, vsebinski del, glavni meni in noga. Pravzaprav pri tem govorimo o razporeditvi spletne strani (ang. Layout), ki pomembno vpliva na kakovost ter uporabnost strani. V nadaljevanju bomo spoznali več vrst razporeditev ter en način, s katerim razporeditve izdelamo. Spletne strani so običajno organizirane v več stolpcev, vsebujejo glavo z logotipom in naslovom ter nogo, s čimer so vsebine strukturirane podobno kot v revijah in časopisih. Takšno strukturo lahko dosežemo s pomočjo tabel ali DIV elementov. Pri razporeditvi pa pomembno vlogo odigrajo tudi CSS stili, s katerimi natančno določimo postavitev posameznih delov HTML strani.

    DIV element

    DIV element se uporablja za združevanje ostalih elementov v nekakšen okvir, ki ga kasneje oblikujemo ter razporejamo zgolj s pomočjo CSS stilov. Gre za element, ki je vedno bolj pomemben pri izdelavi spletnih strani in predvsem pri razporejanju ostalih elementov na spletni strani izpodriva tabele. 

    Za definicijo DIV elementa uporabimo začetno označbo <div> ter končno označbo </div>. V začetni označbi običajno uporabljamo le dva atributa - class in id, s pomočjo katerih določimo zgolj stil elementa.

    Element DIV se od elementa p (paragraf) razlikuje po namenu - DIV se uporablja za združevanje elementov v logične okvire in razporeditev, medtem ko p element služi kot vsebnik besedila. V osnovi bi ju lahko zamenjevali in bi spletne strani zgledale enako, a bi iskalniki (Google, Bing...) našo stran nepravilno zabeležili v svoj imenik spletnih strani.
